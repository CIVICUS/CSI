CIVICUS CSI README

June 2013
Nina Mian

This is a repository set up to store and share primary research data from Civil Society Index Phase 2, developed by CIVICUS: World Alliance for Citizen Participation 

For more information on CIVICUS please visit https://civicus.org/

This package contains:
1. README (this file)
2. datapackage.json (json description of the dimensions in csi_phase2_33_countries.csv)
3. csi_phase2_33_countries.csv (comma separated value file of the values for each dimension and sub-dimension)
4. csi_pahse2_33_countries.xls (excel version of the dimension titles and values)
5. Overview of Dimensions (Word doc describing the CSI dimensions)
